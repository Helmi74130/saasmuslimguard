# Guide de démarrage rapide

Installez et configurez MuslimGuard en **moins de 5 minutes** avec ce guide rapide.

## Prérequis

<Callout type="info">
Avant de commencer, assurez-vous d'avoir :
- **Google Chrome** version 100 ou supérieure
- Un **compte MuslimGuard** ([Créer un compte](/sign-up))
- Une **connexion internet** active
</Callout>

## Étape 1 : Installer l'extension

<Tabs defaultValue="chrome-store">
  <TabsList>
    <TabsTrigger value="chrome-store">Chrome Web Store</TabsTrigger>
    <TabsTrigger value="manual">Installation manuelle</TabsTrigger>
  </TabsList>

  <TabsContent value="chrome-store">
    ### Installation depuis le Chrome Web Store

    1. Visitez le [Chrome Web Store](https://chrome.google.com/webstore)
    2. Recherchez **"MuslimGuard"**
    3. Cliquez sur **Ajouter à Chrome**
    4. Confirmez l'installation en cliquant sur **Ajouter l'extension**

    <Callout type="success">
    L'installation prend environ 10 secondes. Vous verrez l'icône MuslimGuard apparaître dans votre barre d'outils.
    </Callout>
  </TabsContent>

  <TabsContent value="manual">
    ### Installation manuelle (Développeurs)

    Pour les développeurs ou si vous testez une version locale :

    ```bash
    # Cloner le repository
    git clone https://github.com/muslimguard/extension.git
    cd extension

    # Installer les dépendances
    pnpm install

    # Build l'extension
    pnpm build
    ```

    Puis :
    1. Ouvrez `chrome://extensions`
    2. Activez le **Mode développeur**
    3. Cliquez sur **Charger l'extension non empaquetée**
    4. Sélectionnez le dossier `dist/`
  </TabsContent>
</Tabs>

## Étape 2 : Lier votre compte

Après l'installation, l'extension doit être liée à votre compte MuslimGuard.

### Processus de liaison

1. Cliquez sur l'**icône MuslimGuard** dans votre barre d'outils
2. Cliquez sur **Se connecter**
3. Entrez vos identifiants MuslimGuard
4. Autorisez l'extension à accéder à votre compte

```javascript
// L'extension appelle automatiquement l'API
await fetch('/api/extension/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    deviceName: 'Chrome - Windows 10',
    extensionVersion: '1.0.0'
  })
});
```

<Callout type="success">
Une fois liée, l'extension synchronise automatiquement votre statut d'abonnement et vos paramètres !
</Callout>

## Étape 3 : Configuration de base

### Configurer les heures de prière

1. Ouvrez l'extension
2. Allez dans **Paramètres > Heures de prière**
3. Sélectionnez votre ville ou entrez manuellement
4. Activez la **Pause automatique**

### Ajouter votre première règle de blocage

1. Allez dans **Paramètres > Blocage**
2. Cliquez sur **Ajouter un site**
3. Entrez l'URL à bloquer (ex: `youtube.com`)
4. Cliquez sur **Enregistrer**

<Callout type="warning">
Les règles de blocage prennent effet immédiatement. Assurez-vous de tester sur un onglet de navigation privée.
</Callout>

## Étape 4 : Vérification

Testez que tout fonctionne correctement :

### Test de blocage

1. Essayez d'accéder à un site bloqué
2. Vous devriez voir une page de blocage MuslimGuard
3. Vérifiez que le compteur de blocages augmente

### Test de l'heure de prière

1. Vérifiez l'heure de la prochaine prière dans l'extension
2. Les 5 minutes avant la prière, un rappel apparaît
3. Pendant la prière, tous les sites sont bloqués
