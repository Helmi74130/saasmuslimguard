# Installation de MuslimGuard

Bienvenue dans le guide d'installation de MuslimGuard. Cette section vous guidera à travers les différentes méthodes d'installation de l'extension.

## Méthodes d'installation disponibles

<Callout type="info">
MuslimGuard peut être installé de deux façons : via le Chrome Web Store (recommandé) ou manuellement pour le développement.
</Callout>

### Installation depuis le Chrome Web Store

La méthode la plus simple et recommandée pour la plupart des utilisateurs. L'extension se met à jour automatiquement.

[Voir le guide détaillé →](/docs/installation/chrome-store)

### Installation manuelle

Pour les développeurs ou pour tester des versions en développement.

[Voir le guide d'installation manuelle →](/docs/installation/manual-installation)

## Prérequis

Avant d'installer MuslimGuard, assurez-vous de disposer de :

- **Navigateur compatible** : Google Chrome version 100 ou supérieure
- **Compte MuslimGuard** : Créez un compte gratuit sur [muslimguard.com](/sign-up)
- **Connexion Internet** : Requise pour la synchronisation et les mises à jour

## Vérification de l'installation

Une fois l'installation terminée, vous devriez voir :

1. L'icône MuslimGuard dans votre barre d'extensions Chrome
2. Un badge de notification vous invitant à vous connecter
3. L'extension listée dans `chrome://extensions`
