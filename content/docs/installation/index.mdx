# Installation de MuslimGuard

Ce guide vous accompagne dans la mise en place de MuslimGuard sur votre navigateur. L'installation est conçue pour être rapide et ne nécessite aucune compétence technique particulière.

## Méthode d'installation recommandée

La diffusion de MuslimGuard s'effectue exclusivement via le canal officiel afin de garantir la sécurité de vos données et la réception automatique des mises à jour de protection.

### Installation depuis le Chrome Web Store

C'est la méthode standard pour tous les utilisateurs. Elle garantit que l'extension est vérifiée et conforme aux normes de sécurité des navigateurs basés sur Chromium.

1. Rendez-vous sur la page officielle de MuslimGuard dans le Chrome Web Store.
2. Cliquez sur le bouton **Ajouter à Chrome**.
3. Validez l'autorisation dans la fenêtre contextuelle qui s'affiche.
4. L'extension s'installe et s'active instantanément.

[Accéder au Chrome Web Store →](https://chromewebstore.google.com/detail/muslimguard-contr%C3%B4le-pare/khilbklefblhhcombalbofjjliplfdnp)

## Prérequis système

Avant de procéder à l'installation, vérifiez les points suivants :

- **Navigateur compatible** : Google Chrome (version 100+), Microsoft Edge, Brave ou tout autre navigateur basé sur Chromium.
- **Compte MuslimGuard** : Une inscription préalable est nécessaire pour synchroniser vos règles de filtrage. [Créer un compte](/sign-up).
- **Accès administrateur** : Assurez-vous d'avoir les droits nécessaires pour ajouter des extensions sur l'ordinateur cible.

## Vérification de l'installation

Une fois le processus terminé, vous pouvez confirmer le bon fonctionnement de l'outil via les indicateurs suivants :

1. **Barre d'outils** : L'icône MuslimGuard apparaît à droite de votre barre d'adresse.
2. **Badge d'activation** : Un indicateur visuel vous invite à vous connecter pour charger votre configuration.
3. **Gestionnaire d'extensions** : MuslimGuard figure dans la liste de vos extensions actives (`chrome://extensions`).

## Configuration post-installation

Une fois l'extension installée, il est impératif de finaliser les réglages pour activer la protection :
