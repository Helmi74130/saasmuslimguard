# Installation depuis le Chrome Web Store

La méthode la plus simple et sécurisée pour installer MuslimGuard est via le Chrome Web Store officiel.

## Étapes d'installation

### 1. Accéder au Chrome Web Store

<Callout type="info">
Assurez-vous d'être connecté à votre compte Google Chrome pour installer l'extension.
</Callout>

Rendez-vous sur la page officielle de MuslimGuard :
- [MuslimGuard sur Chrome Web Store](https://chromewebstore.google.com/detail/muslimguard-contr%C3%B4le-pare/khilbklefblhhcombalbofjjliplfdnp)

### 2. Ajouter à Chrome

1. Cliquez sur le bouton **"Ajouter à Chrome"**
2. Une fenêtre contextuelle apparaîtra listant les permissions requises
3. Lisez attentivement les permissions demandées
4. Cliquez sur **"Ajouter l'extension"** pour confirmer

### 3. Vérifier l'installation

Une fois installée, vous verrez :
-  Une notification de confirmation d'installation
-  Un badge vous invitant à vous connecter
-  L'icône MuslimGuard apparaître dans votre barre d'extensions

## Permissions requises

MuslimGuard demande les permissions suivantes pour fonctionner :

| Permission | Raison | Utilisation |
|------------|--------|-------------|
| **webRequest** | Bloquer les sites web | Intercepter et bloquer l'accès aux sites indésirables |
| **storage** | Sauvegarder vos paramètres | Stocker localement vos règles de blocage |
| **activeTab** | Scanner le contenu | Détecter les mots-clés inappropriés sur les pages visitées |
| **alarms** | Notifications de prière | Déclencher les rappels aux heures de prière |

<Callout type="success">
Toutes vos données restent sur votre appareil. MuslimGuard ne collecte ni ne partage vos informations de navigation.
</Callout>

## Épingler l'extension

Pour un accès facile, épinglez MuslimGuard à votre barre d'outils :

1. Cliquez sur l'icône des extensions (pièce de puzzle) dans Chrome
2. Trouvez "MuslimGuard" dans la liste
3. Cliquez sur l'icône de punaise à côté du nom

## Configuration initiale

Après l'installation, l'extension vous guidera automatiquement pour :

1. **Se connecter à votre compte** - Utilisez vos identifiants MuslimGuard
2. **Lier l'extension** - Associez l'extension à votre abonnement
3. **Synchroniser les paramètres** - Récupérez vos règles de blocage

[Guide de configuration initiale →](/docs/introduction/quick-start)

## Mises à jour automatiques

<Callout type="info">
Les extensions installées depuis le Chrome Web Store se mettent à jour automatiquement. Vous recevrez toujours la dernière version de MuslimGuard sans action de votre part.
</Callout>

### Vérifier la version installée

Pour voir quelle version vous utilisez :

1. Allez dans `chrome://extensions`
2. Activez le "Mode développeur" en haut à droite
3. Trouvez MuslimGuard et consultez le numéro de version

## Désinstallation

Si vous souhaitez désinstaller l'extension :

1. Faites un clic droit sur l'icône MuslimGuard
2. Sélectionnez **"Supprimer de Chrome"**
3. Confirmez la désinstallation

<Callout type="warning">
La désinstallation supprimera vos paramètres locaux. Pour conserver vos paramètres, assurez-vous qu'ils sont synchronisés avec votre compte avant de désinstaller.
</Callout>
