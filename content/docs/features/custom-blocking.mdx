# Blocage personnalisé

Le **blocage personnalisé** est au cœur de MuslimGuard. Contrairement aux solutions de contrôle parental classiques qui imposent des listes génériques, MuslimGuard vous donne le **contrôle total**.

## Pourquoi le blocage personnalisé ?

Chaque famille est différente :

- Les valeurs et la sensibilité varient d'une famille à l'autre
- L'âge des enfants influence ce qui est approprié
- Certains contenus peuvent être acceptables pour certains, pas pour d'autres

<Callout type="success">
**Vous décidez ce qui est approprié pour votre famille. MuslimGuard applique vos règles.**
</Callout>

## Ce que vous pouvez bloquer

### 1. Sites web spécifiques

Bloquez des sites entiers selon vos choix :

- Réseaux sociaux inappropriés
- Sites de jeux en ligne
- Forums non modérés
- N'importe quel site que vous jugez inapproprié

### 2. Catégories de contenus

Définissez des catégories complètes à bloquer :

- Contenus adultes et pornographie
- Gambling et paris en ligne
- Violence excessive
- Contenus contraires aux valeurs islamiques
- Sites de rencontre
- Contenus haineux ou toxiques

### 3. Règles personnalisées

Créez vos propres règles de navigation :

- Bloquer certains types de vidéos
- Restreindre l'accès à certaines plateformes
- Limiter les achats en ligne
- Bloquer les publicités inappropriées

## Comment ça fonctionne ?

### Étape 1 : Accédez aux paramètres

Depuis votre **tableau de bord MuslimGuard**, accédez à la section "Blocage personnalisé".

### Étape 2 : Ajoutez vos règles

Vous pouvez ajouter :

- **URL complètes** : `inappropriate-site.com`
- **Domaines entiers** : `*.bad-domain.com`
- **Mots-clés dans les URLs** : Bloquez toute URL contenant certains termes

### Étape 3 : Organisez par profils

Créez différents profils pour chaque enfant :

- **Profil 8-12 ans** : Blocage strict
- **Profil 13-16 ans** : Blocage modéré avec plus de liberté
- **Profil 16+** : Blocage ciblé sur le haram

<Callout type="info">
Chaque enfant peut avoir ses propres règles adaptées à son âge et sa maturité.
</Callout>

## Avantages du blocage personnalisé

###  Pas de listes imposées

- Vous ne dépendez pas de listes génériques qui peuvent être trop strictes ou trop laxistes
- Vous ajustez selon vos valeurs propres

###  Contrôle total

- Vous savez exactement ce qui est bloqué
- Vous pouvez débloquer ou bloquer à tout moment
- Vous gardez la main sur l'éducation numérique de vos enfants

###  Évolution avec vos enfants

- Les règles s'adaptent à la croissance de vos enfants
- Vous pouvez assouplir ou renforcer selon les besoins
- Flexibilité totale

###  Respect de vos principes islamiques

- Bloquez ce qui est clairement haram
- Ajustez selon votre école de pensée
- Préservez vos valeurs familiales

## Exemples d'utilisation

### Exemple 1 : Famille avec jeunes enfants

```
Sites bloqués :
- Tous les réseaux sociaux
- Sites de streaming non contrôlés
- Forums et chat en ligne
- Sites de jeux multijoueurs

Catégories bloquées :
- Contenus adultes (strict)
- Violence
- Langage inapproprié
```

### Exemple 2 : Famille avec adolescents

```
Sites bloqués :
- Sites de rencontre
- Gambling
- Contenus adultes explicites

Catégories bloquées :
- Pornographie
- Paris en ligne
- Contenus haineux

Autorisations :
- YouTube avec SafeSearch activé
- Réseaux sociaux avec supervision
```

## Meilleures pratiques

<Callout type="warning">
**Important :** Communiquez avec vos enfants sur les raisons du blocage. L'éducation est aussi importante que la protection technique.
</Callout>

### 1. Commencez strict, puis assouplissez

Il est plus facile de débloquer progressivement que de bloquer après exposition.

### 2. Révisez régulièrement

- Vérifiez vos règles tous les 3-6 mois
- Adaptez selon l'âge et la maturité
- Écoutez les retours de vos enfants

### 3. Combinez avec d'autres fonctionnalités

Le blocage personnalisé fonctionne encore mieux avec :

- [Détection temps réel](/docs/features/keyword-detection) pour les contenus non répertoriés
- [Contrôle horaire](/docs/features/time-controls) pour limiter le temps d'écran
- [Mode prière](/docs/features/prayer-mode) pour les obligations religieuses

## Limitations à connaître

- Le blocage fonctionne uniquement sur les navigateurs où l'extension est installée
- Les utilisateurs techniques peuvent tenter de contourner (d'où l'importance de l'éducation)
- Nécessite une mise à jour régulière des règles
